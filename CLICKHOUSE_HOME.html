ClickHouse 数据库简介

OLAP Columnar High Performance

ClickHouse 是一个开源列式数据库，专注于在线分析处理（OLAP）场景， 适合海量数据的实时查询、统计分析和日志检索。

核心特点

  • 列式存储：仅读取需要的列，显著降低 I/O。
  • 高压缩比：结合编码与压缩算法，节省存储成本。
  • 向量化执行：对 CPU 更友好，聚合查询速度快。
  • 分布式能力：支持分片与副本，横向扩展方便。
  • SQL 友好：支持丰富函数与聚合能力。

典型场景

  • 用户行为分析与埋点报表
  • 日志与可观测性数据查询
  • 业务指标看板与实时报表
  • 广告、风控与运营分析

快速示例

建表：

CREATE TABLE events(ts DateTime, user_id UInt64, action String) ENGINE = MergeTree ORDER BY (ts, user_id);

查询：

SELECT action, count() FROM events WHERE ts > now() - INTERVAL 1 DAY GROUP BY action ORDER BY count() DESC;
